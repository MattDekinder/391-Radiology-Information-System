Report generate:

  1. The user may select a start date for the search results to begin greater than or equal to.
  2. The user may select an end date for the search results to begin less than or equal to.
  3. Enter a diagnosis to search for into the text box. This is case sensitive.
  4. select submit to begin the search and the results, if any, will be displayed in a table below the search bar.
  5. select back to return to the search page
