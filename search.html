Search:

  1. Depending on your user class: buttons will appear along the top
  1. Everyone can see the logout button which will log the user out and return to the login screen
  2. Radiologists can see the Upload button and select it to navigate to the upload module
  3. Administrators can see the User Management button and use it to navigate to the user management module
  4. Administrators can see the Data Analysis button and use it to navigate to the data analysis module
  5. Administrators can see the Report Generate button and use it to navigate to the report generating module
  1. The user may enter a series of search strings separated by “and” into the search box. The search is not case sensitive.
  2. the user may select the method for sorting the search results by the option menu (most relevant is sorting by score returned by oracle after the provided weight function is applied)
  3. The user may select a start date for the search results to begin greater than or equal to.
  4. The user may select an end date for the search results to begin less than or equal to (the date functions are applied to both prescribing date and testing date and will return a result if either pass the condition).
  5. select the search button to begin the search with the selected options.
